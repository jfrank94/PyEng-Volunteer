1) Use at least 32GB to install OpenCV3 for Ubuntu 16.04
2) The install will need much processing power, so 
if you’re not using a PC (non-laptop) with an excellent processor and a fan, I’d suggest to leave your computer alone while installing the packages.
3) These were my only barriers to installing OpenCV3 on a virtual machine. I used the following instructions to install OpenCV3 with ffmpeg enabled: https://github.com/BVLC/caffe/wiki/OpenCV-3.1-Installation-Guide-on-Ubuntu-16.04
